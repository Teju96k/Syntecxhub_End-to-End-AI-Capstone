An AI-powered desktop assistant developed in Python that authenticates users through face detection, accepts voice and text commands, and performs various desktop automation tasks.

---

## 📌 Features

- 🔐 Face Authentication
- 🎤 Voice Command Recognition
- 💬 Text-Based Commands
- 🗣 Text-to-Speech Response
- 🖥 Open System Applications
- 🌐 Google Search
- 📅 Date & Time Information
- 🪟 User-Friendly Tkinter GUI

---

## 🛠 Technologies Used

- Python 3.14
- Tkinter
- OpenCV
- SpeechRecognition
- sounddevice
- pyttsx3
- NumPy
- Pillow
- SciPy

## ⚙ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/End-to-End-AI-Capstone.git
```

Open the project folder:

```bash
cd End-to-End-AI-Capstone
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python main.py
```
